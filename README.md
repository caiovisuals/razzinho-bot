## Razzinho Bot

Bot do Discord pro servidor World of Razzar! Feito com muita dedicação pro meu parceiro Rafael.

## Comandos
 
| Comando | Descrição |
|---|---|
| `!ping` | Mostra a latência do bot e da API do Discord |
| `!sorteio` | Sorteia um membro aleatório do servidor |
| `!frasedodia` | Envia uma frase motivacional aleatória |
| `!say` | Repetir a mensagem que o usuário escrever |
| `!help` | Lista todos os comandos disponíveis |

## Objetivo

- Estudar o desenvolvimento de bots para Discord com JavaScript
- Aprender a usar a biblioteca discord.js e suas funcionalidades
- Praticar organização de projetos com carregamento dinâmico de módulos
- Explorar o uso de variáveis de ambiente com dotenv
- Criar uma experiência divertida e interativa pro servidor World of Razzar

## Dependências

- **[dotenv] - ^16.4.5
